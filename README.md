# Pasillo-y-curva
Este programa, desarrollado mediante scripting en Python dentro de Blender Foundation (Blender), tiene como objetivo generar de forma automática un escenario 3D compuesto por un camino con tramos rectos y curvos, acompañado de muros laterales, materiales personalizados y una cámara animada que recorre el trayecto de manera suave y continua
